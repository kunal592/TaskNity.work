# **App Name**: TaskNity.Work Frontend

## Core Features:

- Project Display: Display a list of projects with their titles, members, and progress using mock data.
- Animated Navbar: A premium navbar component with GSAP animation for smooth transitions.
- Dashboard Home Page: Overview of all projects in a grid layout, displaying the project title and progress, and using framer-motion for loading transitions.
- Authentication Simulation: A UI context for simulating user authentication using the mock data.

## Style Guidelines:

- Primary color: Gradient from purple (#9333EA) to pink (#EC4899) to red (#F44336), creating a modern, vibrant feel in line with current design trends.
- Background color: Light gray (#F9FAFB), provides a clean, neutral backdrop that allows the primary colors and content to stand out.
- Accent color: Teal (#008080), offering a fresh, distinctive alternative.
- Body and headline font: 'Inter', a grotesque sans-serif known for its modern, machined, neutral appearance, offering both clarity and style to headings and body text. Note: currently only Google Fonts are supported.
- Subtle, smooth animations on the Navbar (GSAP) and cards (framer-motion) to enhance user experience.
- Clean, grid-based layout for project display to provide clear information hierarchy.
- Lucide React icons for a modern and consistent UI.
